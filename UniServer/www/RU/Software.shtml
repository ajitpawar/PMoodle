Free Software You can install
Software installation 

You can install the programs from the Extra folder on the Cd Rom....

A) Foxit PDF Viewer - used to look at PDF documents  PDF manual here
B) FireFox Web Browser - web browser - portable version - just copy folder to your HD
C) Install MS Word Viewer - if you do not have other software to open MS Word files
D) Install MS PowerPoint Viewer – if you do not have other software to open Power Point Files
E) Install MS Excel Viewer -  if you do not have other software to open Excel documents
F) DeepBurner Portable– if you do not have other CD making software - just copy folder to your HD
G) Media Player Classic - used to play all kinds of media files - just copy folder to your HD H) Install FireFox Windows Media Plugin – if you want to play windows media files with FireFox
