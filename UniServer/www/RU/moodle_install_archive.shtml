To make a new moodle install file follow these steps...
You do not need to do this if you want to keep the install the same.
You only need to do this if you want to include your changes inside a new installation of mobile moodle.

1.) Delete the pm1.20.exe in the Install folder
2.) Open IZARC from the menu (or use any other zip software you have)
3.) Select all the files in the root folder and the udrive folder (do not include the extra or moodledocs_en folder)
4.) Add these to an archive named pm1.20.zip in the Install folder
5.) Make the pm1.20.zip file a self executing archive using IZARC
6.) You can copy the information from Install Instructons.txt when it comes time to fill in the information in the description of the new exe file.  You can also write whatever else you want instead.
7.) Enter the default folder you want the self executing archive to install to.  We suggest C:\pm1.20 but you can make it whatever is convenient for you.
