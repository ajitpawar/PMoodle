To Create Intranet Server

There is only one file you have to edit make this install into an "full on" intranet server. The \udrive\www\moodle\config.php.

Here is the line:

$CFG->wwwroot   = 'http://localhost:4001/moodle'; to $CFG->wwwroot   = 'http://myipaddress:4001/moodle';
This would be the ip address of your Portable Moodle machine. Start--> Run--->cmd---> c:\.ipconfig ... copy the ipaddress  to left of the colon. Close and save config.php.

Make sure your firewall has port 4001 open . Go to other machine type in  http://ipaddress:4001/moodle. This will launch moodle. Make sure you have connectivity to your new server you might need to turn off the firewall for testing and use ping to verify.

There is one more file you will need to edit if you want to preserve these settings after changing modes.

\udrive\plugins\uscd2\config_orig.php. If you don't change this file also and you change modes your edit will be overwritten.

The fact that this configuration would work for an internet server is highly discouraged. There is not enough security to properly serve people coming in off the internet. If you decide to use with internet ***USE AT YOUR OWN RISK***.
