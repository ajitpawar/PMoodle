How to Make a new Moodle CD

There are several steps to make a new Moodle CD containing your own courses.
There are two modes of operation. CD mode and US mode(user mode). You must be in US mode to make changes to Moodle. You must be in CD mode to make a new CD. Follow the steps below to make a new Moodle CD.

Install Moodle
If you want to make a new CD you must first install moodle to your hard disk drive. There is a link on the control panel menu to do this.
When you receive the CD it will start by itself if possible. It can only start by itself if autostart is enabled for the CD drive in your computer. When it starts it will be in CD mode. In this mode things added to moodle are stored on the hard disk of the computer in a folder named C:\uscd35temp. These additions to moodle will be lost if a new CD is made.
The install will copy moodle to C:\UniServer on your hard disk. From there you can start moodle by running Server_Start.bat.

Make Changes to Moodle
When you have installed moodle on the hard disk you can add new courses or make whatever other changes you want. Before you make a new CD you must return to the CD mode. The CD you make will not work if you do not do this.

Change to CD Mode
To change to CD mode you must run C:\Uniserver\mode_cd.bat

Make the new Moodle CD
After you have done this you can make a new CD. There is a CD making software called Deepburner on the original CD. You can use this software to make the new CD. If you have your own software to make CD's you can use that. To make the CD copy the files from C:\UniServer on your hard disk to the CD.

Change Moodle back to US Mode
You should change Moodle back to the US mode. This will make it easier to make more changes later.
Follow the steps below...
Run C:\Uniserver\mode_us.bat
You do not have to shut down the server. The mode_us.bat command will do everything for you.
