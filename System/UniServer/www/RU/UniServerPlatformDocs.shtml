UniServer Platform Documentation

The people at Uniformserver have made a wonderful portable LAMP server.
Find them here http://www.uniformserver.com/
They have also written great documentation. http://center.uniformserver.com/index.html
Some documentation is copied below if you do not have an internet connection.

UNIFORM Server from CD1.PDF Explains in detail how to prepare the server software for use on a CD. We made modifications to the batch files and perl pl files for use with Moodle. Please examine those files to learn the details.
Unicenter Batch file snippets.PDF Explains how the Batch files are used to start and stop Uniserver.
Moodle Install.PDF Explains how to install moodle on Uniserver.
Uniserver Known Problems.PDF Explains Uniserver problems we know about.
Wikka Wiki on Uniform Server.PDF Tells how you can install this wiki software. You will have to do this on your own. We have not done this here.
